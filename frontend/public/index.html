# AI CV Analyzer Frontend This is the frontend application for the AI CV Analyzer project. It provides a modern, responsive interface for uploading, analyzing, and managing CVs with AI-powered insights. ## Getting Started 1. Install dependencies: ```bash npm install ``` 2. Start the development server: ```bash npm run dev ``` 3. Open [http://localhost:5173](http://localhost:5173) in your browser ## Features - Modern, responsive UI built with Tailwind CSS - CV upload with drag-and-drop functionality - AI-powered CV analysis and scoring - Dashboard with statistics and insights - Detailed CV analysis reports - Search and filtering capabilities - User authentication system ## Tech Stack - React Router v7 - TypeScript - Tailwind CSS - Vite - Lucide React ## Project Structure - `app/components/` - Reusable UI components - `app/lib/` - Utility functions and API client - `app/routes/` - Page components - `types/` - TypeScript type definitions - `constants/` - Application constants ## API Integration The frontend is designed to work with a backend API. Configure the API base URL in your environment variables: ```env VITE_API_BASE_URL=http://localhost:8000 ``` ## Building for Production ```bash npm run build ``` The built files will be in the `build/` directory.
